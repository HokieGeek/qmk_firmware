### [TMUX](#feature-tmux)
| keycode | action |
|---|---|
| `TMUX_NEXT` | Traverses to next window |
| `TMUX_PREV` | Traverses to previous window |
| `TMUX_LAST` | Switches to last active window |
| `TMUX_PEEK` | Momentarily switches to last active window while key is down |
